## What is Slate?

A dead simple markdown editor for macOS. No account, no sync, no cloud. 
Just you and your files.

Think in markdown. Reason with AI.

→ [slatemd.app](https://www.slatemd.app)

---

## Installation

1. Download `Slate.dmg` from the latest release
2. Open the DMG and drag Slate to your Applications folder
3. Click "Open" in slate and select your first MD workspace folder.

## System Requirements

- macOS 10.13 or later
- Apple Silicon or Intel Mac (universal binary)
